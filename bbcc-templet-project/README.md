# \# BBCC Templet - Modern Education Management System

# 

# Yeh project ek full-stack web application hai jo education management ke liye banaya gaya hai. Yeh glassmorphism design, dark mode, aur smooth animations ka use karta hai for a modern digital look. Yeh multi-role login (Admin, Teacher, Student), CMS toggles, notifications, aur automated reminders include karta hai.

# 

# \## Features

# \- \*\*Multi-Role Login\*\*: Admin, Teacher, aur Student ke liye separate dashboards.

# \- \*\*Admin CMS Panel\*\*: Website sections (Gallery, Courses, Admission) ko toggle karo.

# \- \*\*Teacher Dashboard\*\*: Student list dekho aur real-time notifications bhejo.

# \- \*\*Student Dashboard\*\*: Fee status, attendance dekho, aur popups receive karo.

# \- \*\*Real-Time Notifications\*\*: Socket.io se popups for instant alerts.

# \- \*\*Automated Reminders\*\*: Fee reminders simulate karo (WhatsApp/SMS ke liye API placeholders).

# \- \*\*Modern UI\*\*: Glassmorphism, dark/light mode, aur animations.

# 

# \## Tech Stack

# \- \*\*Frontend\*\*: HTML5, CSS3, JavaScript (Vanilla).

# \- \*\*Backend\*\*: Node.js with Express.

# \- \*\*Database\*\*: MongoDB (Mongoose for schemas; simulated in code for simplicity).

# \- \*\*Real-Time\*\*: Socket.io.

# \- \*\*Styling\*\*: Glassmorphism with CSS blur effects.
